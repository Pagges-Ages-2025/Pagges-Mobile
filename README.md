# Pagges Mobile

Pagges mobile repository 

## Getting started

- Instalar [Expo GO](https://expo.dev/go) no dispositivo mobile 

- Instalar [Node.js](https://nodejs.org/en)

- Instalar [VSCode](https://code.visualstudio.com/)

## Clonar repositório

- Criar uma pasta para o repositório 

- Abrir um terminal dentro da pasta e clonar o repositporio ```git clone https://tools.ages.pucrs.br/pagges/pagges-mobile.git```

## Rodar o Projeto 

- Entrar na pasta ```pagges-mobile``` e rodar o comando ```npx expo start``` 

- Ler o QRCode com o celular. No android usar Expo Go > Scan QR. No ios usar câmera padrão
